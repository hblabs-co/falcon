Projektbeschreibung

Für unseren Kunden aus dem Finanzumfeld in Frankfurt am Main suchen wir eine/n Angular Entwickler (m/w/d). Zu den Hauptaufgaben gehört die (Weiter-) Entwicklung von Webfrontends mit Angular sowie die Integration von Frontendkomponenten mit Backend Services via REST APIs. Darüber hinaus fürhst du Fehleranalysen und -Korrekturen durch und entwickelst automatisierte Frontendtests.

must-have:
-Sehr gute Kenntnisse mit Angular und TypeScript
-Knowhow im Bereich Semantic Versioning und mit Git
-Gute Kenntnisse mit HTML, CSS und JavaScript
-Erfahrung im Umgang mit REST-APIs und mit Paketmanagern wie npm
-Erfahrung im Testing (vorzugsweise mit Playwright, Karma oder Jest)
-Fließende Deutschkenntnisse und Bereitschaft 3 Tage pro Woche vor Ort in FFM zu arbeiten

nice-to-have:
-Erfahrung mit Java und GitHub Actions

Start: asap
Dauer: 6 Monate+
Einsatzort: Frankfurt am Main
